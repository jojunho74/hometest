테이블 설정

Supabase에 alio_jobs 테이블이 없으면 아래 SQL을 실행하세요

📋 테이블 생성 SQL

1. Supabase → 프로젝트 → SQL Editor
2. 아래 SQL을 복사해서 실행하세요 (최초 1회) 

CREATE TABLE IF NOT EXISTS alio_jobs (
  id            UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  dataset_name  TEXT NOT NULL,
  grade         TEXT,
  grade_key     TEXT,
  job           TEXT NOT NULL,
  region        TEXT,
  s1_pass       INTEGER,
  s1_total      INTEGER,
  s2_pass       INTEGER,
  s2_total      INTEGER,
  s3_pass       INTEGER NOT NULL,
  s3_total      INTEGER,
  ratio         DECIMAL(8,2) NOT NULL,
  created_at    TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

ALTER TABLE alio_jobs ENABLE ROW LEVEL SECURITY;

CREATE POLICY "anon read" ON alio_jobs
  FOR SELECT USING (true);

CREATE POLICY "auth insert" ON alio_jobs
  FOR INSERT WITH CHECK (true);

CREATE POLICY "auth delete" ON alio_jobs
  FOR DELETE USING (true);

CREATE POLICY "auth update" ON alio_jobs
  FOR UPDATE USING (true);
복사됨!
