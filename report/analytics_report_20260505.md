### 강점 (Strengths)
1. **월간 2,843 방문** — 운영 30일 기준으로 안정적인 트래픽 확보
2. **네이버 블로그/카페 채널 활성화** — 자연 유입 기반 구축
3. **university-board.html 체류시간 123초** — 핵심 콘텐츠 페이지 참여도 최고
4. **competition.html 1,085건** — 경쟁률 정보가 핵심 집객 콘텐츠로 기능
5. **내부이동 13.9%** — 사이트 내 탐색 흐름이 자연스럽게 이어짐

1. **Google SEO 강화** — 현재 Google 유입 0.3% (8건)로 극히 낮음. 메타태그, 구조화 데이터 보완 시 대형 트래픽 채널 확보 가능

3. **index.html 체류시간 12초** — 메인 히어로 섹션에서 목적 페이지로 빠르게 유도하는 CTA 개선 필요
