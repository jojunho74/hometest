# G-TELP IBT 대학일자리플러스센터 패키지 기획서

> 작성 기준: 2026년 4월 | 대상: 대학일자리플러스센터 담당자

---

## 1. 기획 배경

### 왜 패키지인가

현재 사이트는 "IBT 무료진단 있습니다 → 신청하세요" 구조입니다.  
담당자 입장에서는 **"우리 프로그램이랑 어떻게 연결하지?"** 라는 질문이 해결되지 않으면 신청으로 이어지지 않습니다.

조사 결과, 대학일자리플러스센터는 연간 3가지 핵심 니즈를 가집니다.

| 니즈 | 시기 | IBT 연결 가능성 |
|---|---|---|
| 역량 측정 도구 부족 | 연중 (3월·9월 집중) | ★★★ |
| 정부 지원 실적 데이터 필요 | 학기 초 (3월·9월) | ★★★ |
| 공기업 취업 패키지 영어 파트 공백 | 3월·9월 패키지 시즌 | ★★★ |

이 3가지 니즈에 각각 맞춘 패키지로 IBT를 포지셔닝합니다.

---

## 2. 패키지 구조 전체도

```
┌─────────────────────────────────────────────────────┐
│              G-TELP IBT 대학 지원 패키지              │
│                                                     │
│  [패키지 A]      [패키지 B]       [패키지 C]          │
│  역량측정 패키지  점프업 실적 패키지  공기업 영어 패키지   │
│                                                     │
│           ↓ 모두 동일한 IBT 신청 접수 연결 ↓           │
└─────────────────────────────────────────────────────┘
```

3가지 패키지는 **진입 맥락과 제공 형태가 다를 뿐**, 핵심 서비스(IBT 무료 진단)는 동일합니다.

---

## 3. 패키지 A — 역량측정 패키지

### 타깃 담당자
AI역량평가, 직무적성검사 등 역량 측정 프로그램을 이미 운영 중인 센터.  
영어 진단만 빠져 있거나, 외부 의뢰 비용이 부담인 경우.

### 핵심 메시지
> "직무적성 + 영어진단, 한 번에 완성하세요."  
> 새 프로그램을 추가하는 게 아니라 **기존 역량 측정을 완성**하는 것입니다.

### 제공 내용

| 항목 | 내용 |
|---|---|
| IBT 무료 진단 | 최대 200명, 전액 무료 |
| 제공 데이터 | 학번·이름·점수 포함 엑셀 파일 |
| 데이터 형식 | 기존 역량측정 결과와 병합 가능한 컬럼 구성 제공 |
| 운영 방식 | 코드 배포만, 이후 모든 진행은 G-TELP 처리 |

### 담당자 설득 포인트
- 별도 강사 섭외 없음
- 기존 역량검사 옆에 영어 점수가 추가되는 구조
- 학생은 한 번 신청으로 두 가지 역량 확인 가능

### 최적 신청 시기
- **2월 말 ~ 3월 초** (1학기 개강 직전)
- **8월 말** (2학기 개강 직전)

### 사이트 추가 문구 (안)
```
AI역량평가 운영 중이신가요?
영어 진단까지 한 번에 — G-TELP IBT를 무료로 연계하세요.
직무적성 + 영어 점수를 통합한 역량 데이터를 엑셀로 드립니다.
```

## 4. 패키지 B — 점프업 실적 패키지

### 타깃 담당자
고용노동부 지원 재학생 맞춤형 고용서비스(점프업) 운영 센터.  
실적 보고가 의무이며, 참여자 역량 측정 데이터가 필요한 경우.

### 핵심 메시지
> "점프업 참여자 역량 데이터, IBT로 채우세요."  
> **보고서에 바로 쓸 수 있는 데이터**를 무료로 제공합니다.

### 제공 내용

| 항목 | 내용 |
|---|---|
| IBT 무료 진단 | 최대 200명, 전액 무료 |
| 제공 데이터 | 학번·이름·점수·수준 등급 포함 엑셀 |
| 데이터 활용 | 점프업 실적 보고서 직접 활용 가능 |
| 운영 방식 | 코드 배포만, 이후 모든 진행은 G-TELP 처리 |

### 담당자 설득 포인트
- 무료 + 데이터 제공 = 실적 보고 부담 감소
- 참여자 영어 역량 측정 → 사전/사후 비교 가능
- 정부 지원 프로그램과 자연스럽게 연계

### 최적 신청 시기
- **1월 ~ 2월** (학기 기획 시즌, 가장 중요)
- **8월 ~ 9월** (2학기 점프업 기획 시즌)

### 사이트 추가 문구 (안)
```
점프업 프로그램 운영하세요?
참여자 영어 역량 측정 결과를 엑셀로 드립니다.
보고서에 바로 쓸 수 있는 데이터 — 무료입니다.
```

## 5. 패키지 C — 공기업 영어 패키지

### 타깃 담당자
상반기(3월)·하반기(9월) 공기업 취업 패키지를 운영하는 센터.  
공기업 준비생의 영어 수준 파악이 필요하거나, 패키지에 영어 파트가 빠진 경우.

### 핵심 메시지
> "공기업 취업 패키지 시작 전, IBT로 영어 수준부터 파악하세요."  
> **200개+ 공공기관 인정 시험**으로 실전 준비까지 됩니다.

### 제공 내용

| 항목 | 내용 |
|---|---|
| IBT 무료 진단 | 최대 200명, 전액 무료 |
| 제공 데이터 | 학생별 점수 + 공기업 합격 기준점수 비교 안내 |
| 포지셔닝 | "공기업 영어 준비 사전 진단"으로 패키지 시작점 역할 |
| 운영 방식 | 코드 배포만, 이후 모든 진행은 G-TELP 처리 |

### 담당자 설득 포인트
- 공기업 취업 패키지의 영어 파트를 무료로 완성
- 강사 섭외 없이 영어 수준 데이터 확보
- 학생은 실제 공기업 시험과 동일 난이도로 사전 점검 가능

### 최적 신청 시기
- **2월 말** (상반기 공기업 패키지 오픈 직전)
- **8월 말** (하반기 공기업 패키지 오픈 직전)

### 사이트 추가 문구 (안)
```
공기업 취업 패키지 운영하세요?
패키지 시작 전 IBT 진단으로 학생 영어 수준을 파악하세요.
200개+ 공공기관 인정 시험 — 실전 연습까지 한 번에.
```

## 6. 사이트 적용 구조 기획

### 신규 섹션 위치
기존 "핵심 혜택" 섹션 하단, "진행 방법" 섹션 상단에 삽입.

### 섹션 제목 (안)
```
우리 센터에 맞는 프로그램을 선택하세요
```

### UI 구조
```
[ 역량측정 운영 중 ]  [ 점프업 운영 중 ]  [ 공기업 패키지 운영 중 ]
        ↓ 탭 선택 시 해당 패키지 내용 표시 ↓
  - 핵심 메시지
  - 제공 내용 3가지
  - 최적 신청 시기
  - [지금 신청하기] 버튼
```

### 신청 시기 안내 추가
기존 신청 폼 상단에 아래 캘린더 형태 안내 추가.

```
📅 신청 추천 시기
  1~2월   ▶ 1학기 프로그램 기획 중이라면 지금이 최적
  2월 말  ▶ 3월 개강 전 운영을 원하면 최소 2주 전 신청 필수
  8~9월   ▶ 2학기 프로그램 기획 중이라면 지금이 최적
  10~11월 ▶ 하반기 재신청 / 졸업예정자 영어 점수 수요 활용
```
