## 핵심 전략

```
블로그 / 자료실 방문
        ↓
채널 구독 유도 (채용공고 자료실 페이지 내 버튼)
        ↓
이벤트 참여 → 친구·학과·커뮤니티에 공유
        ↓
신규 방문자 유입 → 재구독 반복
```

## 이벤트 1. "자료 받고 친구한테 공유해요" 챌린지

### 컨셉
> 자료실에서 자료를 받은 김에, 친구한테도 알려주면 추가 혜택을 드려요.

### 진행 방식

| 항목 | 내용 |
|------|------|
| **참여 방법** | 채널 추가 → 자료실 링크를 카카오톡 오픈채팅·학과 단톡방·에브리타임에 공유 → 캡처 전송 |
| **혜택** | 공유 인증 시 다음 달 채용공고 자료 PDF 패키지 선발송 |
| **대상** | 자료실 방문 대학생, 블로그 유입 취준생 |
| **기간** | 월별 선착순 50명 |

### 채널 메시지 예시
```
📢 [지텔프에듀] 친구한테 알려주면 자료 더 드려요!

채용공고 영어 기준 자료
이미 받아보셨나요?

학과 단톡방이나 에브리타임에 공유하고
다음 달 자료 패키지 먼저 받아가세요 📋

✅ 참여방법
1. 채널 추가
2. 자료실 링크 단톡방·에브리타임에 공유
3. 캡처 → 채팅으로 전송

📅 월 선착순 50명 마감
[채널 추가하기]
```

### 기대 효과
- 학과 단톡방·에브리타임 공유로 **대학생 집단 내 바이럴**
- 블로그 방문자 → 채널 구독 → 지인 공유의 **자연 확산 루프** 형성
- 공유 캡처 수집으로 **실제 확산 규모 파악** 가능

## 이벤트 2. "우리 학교도 G-TELP 해요" 제보 이벤트

### 컨셉
> 자기 학교에 G-TELP IBT가 없는 게 아쉬운 학생이 학교 담당자에게 직접 알리도록 유도

### 진행 방식

| 항목 | 내용 |
|------|------|
| **참여 방법** | 채널 추가 → 학교 취업지원센터에 G-TELP IBT 도입 건의 후 캡처(이메일·건의함 등) 전송 |
| **혜택** | 건의 인증 시 G-TELP 기출 자료 또는 응시료 할인 쿠폰 제공 |
| **대상** | 자료실·블로그 방문 재학생 |
| **기간** | 상시 운영 |

### 채널 메시지 예시
```
🏫 [지텔프에듀] 우리 학교에 G-TELP 없어서 아쉬우셨나요?

학교 취업지원센터에 도입 건의하고
혜택 받아가세요!

이메일 한 줄, 건의함 클릭 한 번으로
학교 친구들도 혜택을 받을 수 있어요 🎓

✅ 참여방법
1. 채널 추가
2. 학교 취업지원센터에 G-TELP IBT 도입 건의
   (이메일·학교 건의함·직접 방문 모두 가능)
3. 캡처 → 채팅으로 전송

🎁 혜택: G-TELP 기출 자료 + 응시료 할인 쿠폰
[채널 추가하기]
```

### 기대 효과
- 학생이 대학 담당자에게 **직접 도입 요청** → 대학 유입 채널 확보
- 자료실 방문자가 **자발적 영업사원** 역할
- 건의 데이터로 **도입 수요가 높은 학교 파악** 가능 → 우선 영업 대상 선정

## 공통 운영 팁

- 두 이벤트 모두 **자료실 페이지 내 채널 구독 버튼** 옆에 이벤트 안내 문구 추가 권장
- 블로그 포스팅 하단에 "채널 추가하면 이벤트 참여 가능" 문구 삽입
- 카카오 채널 관리자센터 → 메시지 발송으로 월 1회 리마인드 발송
- 구독자 100명 기준 1회 발송 비용 약 1,500~2,000원
