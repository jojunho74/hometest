클로드코드에서 에이전트를 4명 운영할거야. 관리자페이지에 대활활동데시보드를 만들고, 등록리스트를 만들어. 리스트에는 학교명과 링크를 입력하고, 수정 및 삭제,등록이 가능해야해. 이는 데이터베이스로 등록되어야해. 우측에 4개의 색션을 나누어줘. 섹션의 기능은 추후 결정할게. 다음은 워크플로우를 알려줄게. 하나는 수집에이젼트, 비교에이젼트,게시에이젼트, 자동화에이젼트입니다. 먼저 a.수집에이젼트는 1.사용자가 등록해준 특정사이트로 이동, 2. 이동후 페이지를 스크롤링한다. 3. 데이터베이스를 수집한다. 학교명, 게시글 제목, 링크주소를 저장한다. b.비교에이젼트는 수집에이젼트가 만들어준 데이터를 기존의 데이터와 비교하여, 중복된 제목과 링크는 데이터베이스에 저장하지 않는다. c. 게시에이젼트는 관리자페이지에 대학활동메뉴의 등록폼을 확인하고, 학교명, 제목에는 수집된 게시글제목, 링크를 자동으로 입력한다. d.자동화에이젼트는 n8n을 통해서 시작버튼을 클릭하면 워크플로우를 자동으로 실행해야. 운영체제: Windows
사용 도구: Claude Code, n8n, Python,versel,githube,
API: Claude API (ANTHROPIC_API_KEY 환경변수 사용)
저장: supabase로 데이터베이스화
